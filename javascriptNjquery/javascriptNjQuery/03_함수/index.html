내장 함수

시간에 따른 메소드

  • 시간에 따라 명령을 수행하게 해주는 메소드

    setTimeout(a,b); - a: 함수나 명령어, b: 지연시간(1/1000초 단위) clearTimeout(a); - 내용중지

  • 시간에 따라 반복하여 명령을 수행하게 해주는 메소드

    setInterval(a,b); - a: 함수나 명령어, b: 지연시간(1/1000초 단위) clearInterval(a); - 내용중지

내장 함수

  • parseInt() - 문자형 데이터를 정수형 데이터로 바꿈 (소숫점 없이)
  • parseFloat() - 문자형 데이터를 실수형 데이터로 바꿈 (소숫점 있는)
  • String() - 문자형 데이터로 바꿈
  • Number() - 숫자형 데이터로 바꿈
  • Boolean() - 논리형 데이터로 바꿈
  • IsNaN() - 'Not a Number' 숫자가 아닌 문자를 포함하면 true 값 반환
  • eval() - 문자형 데이터를 큰따옴표가 없는 스크립트 코드로 처리하고 산술연산 해줌
  • alert() - 경고창
  • confirm() - 확인/취소창(true/false)
  • prompt() - 입력 대화상사를 만들어 줌(묻고 답하는)
